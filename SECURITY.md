# Security Policy

## Supported Versions

Security fixes are applied to the latest released version.

## Reporting a Vulnerability

Please do not open a public issue for a suspected security vulnerability.

Report the issue privately to the repository owner with:

- A short description of the vulnerability
- Steps to reproduce, if available
- Affected command, file type, or input source
- Any known workaround

This project processes local Cell Ranger outputs and does not intentionally send
data to external services. Treat input files as sensitive research data and avoid
including sample data in public reports.
